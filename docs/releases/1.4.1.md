# MemeBlip 1.4.1

MemeBlip 1.4.1 fixes local audio imports failing when normal sound files exceeded Axum's previous default multipart body limit.

## Changes

- Supports local audio clips up to 20 MB.
- Supports selecting and importing multiple clips sequentially.
- Rejects empty and oversized files with readable errors.
- Resets the file picker after each import attempt.

The Windows MSI is published as `MemeBlip-Setup.msi` through the automated release workflow.
