# MemeBlip v1.1.0

Install from the landing page:

https://memeblip.dawnlightlabs.com/

## Changes

- Added a normal Windows MSI installer wizard instead of a silent/direct install flow.
- Fixed overlapping installer header text by keeping the WiX banner artwork logo-only.
- Added branded installer artwork using the transparent MemeBlip logo asset.
- Added a first-run VB-CABLE setup gate inside MemeBlip.
- Added VB-CABLE status and setup actions to the Routing page.
- Added a MyInstants-powered sound supplier page.
- Added a `Get more` button in the Sounds page so users can search, preview, and import new sounds.
- Replaced the `Get more` sparkle icon with a sound/radio icon.
- Added native remote sound importing, so MyInstants MP3s are downloaded and stored locally by the companion.
- Added soundboard deletion. Deleting a board moves its clips back to `Meme Kit` instead of deleting the audio files.
- Fixed library play buttons so they preview through the user's default speaker/headphone output instead of silently routing only through the virtual mic path.
- Replaced the hardcoded supplier search text with an animated grey placeholder cycling through famous sound prompts.
- Tray `Close MemeBlip` now clears MemeBlip local ports before exiting.
- Set the MSI product version to `1.1.0`.
- Release workflow now publishes the MSI installer asset used by the landing-page download button.
- README now directs public installs through the landing page.
- VB-CABLE remains separate and is not silently installed by MemeBlip.

## Installer asset

The landing page download button resolves to the latest GitHub Release asset:

https://github.com/amaansyed27/meme-blip/releases/latest/download/MemeBlip-Setup.msi
