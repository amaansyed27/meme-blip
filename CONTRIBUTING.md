# Contributing to MemeBlip

Thanks for helping improve MemeBlip.

## Principles

- Keep features real and testable.
- Do not add dummy controls or placeholder UI.
- Prefer simple layouts over nested panels.
- Keep the dashboard focused on clip playback, soundboards, hotkeys, and audio routing.
- Keep code modular and easy to review.

## Before sending changes

Check that the web dashboard builds, the native companion builds, clip import works, playback works, and active soundboards correctly scope hotkeys.

## Documentation

Update the relevant file in `docs/` when changing architecture, setup, routing, or release behavior.
